# 왕초보 시작 가이드 (정말 쉽게)

컴퓨터·AI가 처음이어도 괜찮습니다. 아래 순서만 **그대로** 따라 하세요.
(영문판: [`Beginner_Guide_EN.md`](./Beginner_Guide_EN.md) · 전체 설명서: [`README.md`](./README.md))

---

## 미리 알아둘 것 (사전 준비물)

- **Windows 컴퓨터**, **인터넷 연결**
- **ChatGPT 계정** (없으면 무료로 만들 수 있어요)
- **Node.js**(무료 프로그램)는 없으면 키트가 **알아서 설치를 도와줍니다**. 미리 깔 필요 없어요.

---

## 1단계. 파일 실행하기

1. `Codex-CLI_One-Click_Kit.bat` 파일을 **두 번 클릭**합니다.
2. **검은 창**이 열리고 한국어 메뉴가 보이면 성공입니다.

> 💡 **파란색 경고창("Windows의 PC 보호")이 떴어요?**
> 정상입니다. 처음 보는 파일이라 뜨는 안내예요. **`추가 정보`** → **`실행`** 을 누르면 됩니다.

## 2단계. 시작하기 (제일 쉬운 방법)

- 메뉴가 보이면 **그냥 `Enter` 키**를 한 번 누르세요. (= 코덱스 시작)
- 키트가 알아서 점검하고, Codex가 없으면 **설치를 도와준 뒤** 켭니다.
  - 설치하겠냐고 물으면 `y` → Enter.

```
    /==================================\
    |      >>>   [  ENTER  ]   <<<      |
    \==================================/
     엔터(Enter) 키 한 번이면 바로 시작!
```

## 3단계. 로그인하기 (처음 한 번만)

1. 인터넷 브라우저가 **저절로 열리며** ChatGPT 로그인 화면이 나옵니다.
2. 평소 쓰던 **ChatGPT 계정**으로 로그인합니다. (무료 계정도 가능)
3. 보안을 위해 **2단계 인증(MFA)** 을 켜두는 것을 권장합니다.

> 로그인 정보는 내 컴퓨터의 `C:\Users\(내이름)\.codex\` 에 저장돼서, 다음부터는 다시 로그인하지 않아도 됩니다.

## 4단계. 시키기 (사용 방법)

- 켜지면 화면에 **"이렇게 시켜보세요"** 예시가 먼저 나옵니다. 그대로 따라 적어도 됩니다.
- 검은 화면에 하고 싶은 일을 **한국어로** 적고 `Enter`:
  - `이 폴더에 어떤 파일이 있는지 알려줘`
  - `이 코드가 무슨 일을 하는지 쉽게 설명해줘`
  - `간단한 메모장 프로그램 만들어줘`
- **끝낼 때**는 `/quit` 을 입력하거나, **`Ctrl` 키와 `C` 키**를 함께 누르세요.

## 모드(안전장치) 이해하기

이 키트는 두 가지 모드가 있어요. 화면 위 "모드 :" 에 지금 모드가 표시됩니다.

| 모드 | 무슨 뜻인가요? |
|---|---|
| **파일수정** (기본) | AI가 **지금 폴더 안에서만** 파일을 만들고 고칠 수 있어요. ("메모장 만들어줘"가 실제로 됩니다.) 폴더 밖·PC 전체는 **절대 못 건드려요.** |
| **읽기 전용** | AI가 파일을 **전혀 안 건드리고** 읽기만 해요. 가장 보수적입니다. |

- 바꾸려면 메뉴 **`5` (모드 바꾸기)**. 한 번 정하면 계속 기억합니다.
- 바탕화면·문서 같은 **개인 폴더**에서 "파일수정"으로 켜면, 키트가 **한 번 더 "계속할까요?"** 라고 물어 안전을 지켜줍니다.

## 메뉴 전체 설명

| 누르면 | 무슨 일이 | 
|---|---|
| **Enter / 1** | 코덱스 바로 시작 |
| **2** | 다른 폴더(바탕화면·문서·새 폴더 등)에서 시작 |
| **3** | 설치 / 다시 설치 |
| **4** | 최신 버전으로 업데이트 |
| **5** | 모드 바꾸기 (파일수정 ↔ 읽기전용) |
| **6** | 안 될 때 — 자동 진단 (`진단결과.txt` 저장) |
| **7** | 제거 (Codex 지우기) |
| **H** | 사용법 보기 · **Q** 끄기 |

## 자주 묻는 것 (FAQ)

**Q. 관리자 권한으로 실행해야 하나요?**
→ 아니요. 그냥 두 번 클릭이면 됩니다.

**Q. 처음에 "Enter=시작 / 읽기만 원하면 R" 이라고 나와요.**
→ 그냥 **Enter**를 누르면 됩니다(파일도 만들 수 있는 추천 모드). 더 보수적으로 읽기만 하고 싶으면 `R`을 누르세요. 한 번만 고르면 계속 기억합니다.

**Q. 설치가 멈춘 것 같아요.**
→ 보통 1~2분 걸립니다. 인터넷에서 받는 중이니 잠시 기다려 주세요.

**Q. `codex` 명령이 안 먹는대요.**
→ 컴퓨터를 **한 번 껐다 켜거나 재시작**한 뒤 다시 실행해 보세요. (방금 설치한 위치를 컴퓨터가 인식하는 데 시간이 필요할 수 있어요.)

**Q. 지우고 싶어요.**
→ 메뉴 **`7`** → `y`. "설정까지 지울까요?"는 보통 **`n`** 을 누르세요(다음에 또 쓸 때 편해요). 되돌리려면 메뉴 `3`으로 다시 설치하면 됩니다.

**Q. 뭔가 막혔어요.**
→ 메뉴 **`6` 진단**을 눌러 화면을 캡처해 두면 원인을 찾기 쉽습니다. 이 키트 파일들은 **함께 한 폴더에 두고** 옮기지 마세요.

## 문제 / 오류 대처 (전체 표)

| 증상 | 원인 (쉽게) | 해결 |
|---|---|---|
| `codex` 인식 못 함 / "command not found" | 설치 위치를 컴퓨터가 아직 모름 (PATH) | **컴퓨터 재시작** 후 다시 실행. 그래도 안 되면 메뉴 `6` 진단 |
| 설치가 실패함 | 인터넷 끊김 / 일시 오류 | 인터넷 확인 후 메뉴 `3`으로 다시 설치 |
| 로그인 창이 안 열림 | 방화벽/네트워크 차단 | 인터넷 확인. 화면에 나온 주소를 직접 브라우저에 붙여넣기 |
| `429` / "Too many requests" | 잠깐 너무 많이 씀 (사용량 한도) | 표시된 시간만큼 기다린 뒤 다시 |
| Node 버전이 낮다고 함 | 오래된 Node.js | <https://nodejs.org> 에서 최신 **LTS**(22 이상 권장) 재설치 |
| 로그인하라는데 막힘 | 계정 보안 설정 | ChatGPT 계정에 **2단계 인증(MFA)** 을 켠 뒤 다시 |
| 한글이 깨져 보임 | 글꼴/인코딩 | 창을 닫고 다시 실행. 그래도면 메뉴 `6` 진단 결과를 캡처 |

> 더 헷갈리면 메뉴 **`6` 진단** 화면을 캡처해서, ChatGPT/클로드 같은 AI에게 붙여넣어 물어보면 원인 찾기가 훨씬 쉬워집니다.

## 자주 쓰는 명령어 (직접 칠 때)

> 보통은 메뉴만 써도 충분해요. 아래는 검은 화면에 직접 입력할 때.

| 명령 | 뜻 |
|---|---|
| `codex` | 대화형으로 시작 (가장 쉬움) |
| `codex "할 일 설명"` | 바로 작업 시키기 — 예: `codex "이 코드 설명해줘"` |
| `codex exec "작업"` | 질문 없이 한 번에 실행 |
| `codex --version` | 설치된 버전 확인 |
| `codex logout` | 로그아웃 |
| `codex --sandbox read-only` | 읽기만 (파일 수정 안 함) |
| `codex --sandbox workspace-write` | 지금 폴더 안에서만 수정 허용 |
| `/help` | (Codex 켜진 상태에서) 사용할 수 있는 기능 보기 |

## 워크플로우 (한눈에)

```
[더블클릭] → [한국어 메뉴] → [Enter = 시작]
   → (처음 한 번) ChatGPT 로그인
   → 작업할 폴더 선택
   → 한국어로 시킨다  →  AI가 답하거나 파일을 만든다
   → /quit 로 종료  →  다시 메뉴로
```

## 파일 위치 / 저장되는 곳

- **이 키트 파일들**: 같은 폴더에 함께 두세요 (`.bat`, README, 가이드, 치트시트, LICENSE).
- **키트 설정**(모드·마지막 폴더 기억): `C:\Users\(내이름)\AppData\Local\codex-cli-one-click-kit\`
